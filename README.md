# Abnormal 1.1.6 简体中文汉化版

> BNPR 法线编辑工具的完整中文翻译版本

## ✨ 版本信息

- **原插件版本**: 1.1.6
- **汉化版本**: v3.0
- **适用 Blender 版本**: 4.5.0+
- **翻译语言**: 简体中文 (zh_CN)
- **发布日期**: 2026-01-04

## 🎯 主要特性

- ✅ **完整翻译**: 154+ 条翻译条目,覆盖所有用户可见文本
- ✅ **标准实现**: 使用 Blender 官方翻译 API (`bpy.app.translations`)
- ✅ **全面覆盖**: N面板、工具栏、搜索菜单、偏好设置全部中文化
- ✅ **专业术语**: 参考 Blender 官方中文版术语标准
- ✅ **调试工具**: 内置翻译调试脚本,方便问题排查

## 📋 翻译内容

### 核心功能
- 启动法线编辑器
- 法线 ↔ 顶点色 转换
- 法线 ↔ 面角属性 转换
- 镜像、展平、对齐、平滑法线
- 球形化法线、法线指向目标
- 筛选遮罩、复制粘贴法线

### 界面元素
- 主面板和工具栏
- 偏好设置 (显示、行为、按键映射)
- 所有操作符和工具提示
- 错误提示和警告信息
