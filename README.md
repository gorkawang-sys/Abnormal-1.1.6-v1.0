# Abnormal 1.1.6 简体中文汉化版

> BNPR 法线编辑工具的完整中文翻译版本

## ✨ 版本信息

- **原插件版本**: 1.1.6
- **汉化版本**: v3.0
- **适用 Blender 版本**: 4.5.0+
- **翻译语言**: 简体中文 (zh_CN)
- **发布日期**: 2026-01-04

## 🎯 主要特性

- ✅ **完整翻译**: 154+ 条翻译条目,覆盖所有用户可见文本
- ✅ **标准实现**: 使用 Blender 官方翻译 API (`bpy.app.translations`)
- ✅ **全面覆盖**: N面板、工具栏、搜索菜单、偏好设置全部中文化
- ✅ **专业术语**: 参考 Blender 官方中文版术语标准
- ✅ **调试工具**: 内置翻译调试脚本,方便问题排查

## 📋 翻译内容

### 核心功能
- 启动法线编辑器
- 法线 ↔ 顶点色 转换
- 法线 ↔ 面角属性 转换
- 镜像、展平、对齐、平滑法线
- 球形化法线、法线指向目标
- 筛选遮罩、复制粘贴法线

### 界面元素
- 主面板和工具栏
- 偏好设置 (显示、行为、按键映射)
- 所有操作符和工具提示
- 错误提示和警告信息

## 🔧 安装方法

### 1. 设置 Blender 语言

**重要**: 必须先设置语言为简体中文!

编辑 → 偏好设置 → 界面
✅ 翻译: 勾选
✅ 语言: 简体中文 (Simplified Chinese)
✅ 工具提示: 勾选
✅ 界面: 勾选


### 2. 安装插件

编辑 → 偏好设置 → 插件 → 安装
选择下载的 Abnormal_CN_1_1_6.zip
✅ 勾选启用 "Abnormal"
保存偏好设置

### 3. 验证安装

打开系统控制台 (Windows: 窗口 → 切换系统控制台),应该看到:

[1] 翻译系统注册成功

[2] 当前 Blender 语言: zh_CN

[2] 翻译条目数: 154（没有应该也没问题）

## 🔍 技术细节

### 实现方式
- 使用 Blender 官方 `bpy.app.translations` API
- 标准翻译字典格式: `{(上下文, "原文"): "译文"}`
- 通配符上下文 `"*"` 确保全面覆盖

### 文件结构
Abnormal/
├── init.py              # 插件入口 + 翻译注册
├── translations.py          # 翻译字典 (154+ 条目)
├── translation_debug.py     # 调试工具
├── operators.py             # 操作符
├── ui.py                    # UI 界面
├── properties.py            # 属性设置
└── ...                      # 其他模块

## 🐛 故障排除

### 翻译未生效?

1. **检查语言设置**: 确认 Blender 语言为 "简体中文"
2. **查看控制台**: 确认显示 "[Abnormal] 翻译系统注册成功"
3. **完全重启**: 禁用插件 → 关闭 Blender → 重启 → 重新启用
4. **运行调试脚本**: 使用插件目录中的 `translation_debug.py`
